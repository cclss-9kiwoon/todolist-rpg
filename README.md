# Todo RPG

Todo RPG는 일상적인 할 일(Todo) 관리에 롤플레잉 게임(RPG) 요소를 결합하여 사용자에게 동기를 부여하는 재미있는 웹 애플리케이션입니다.

## 🌟 주요 기능

- **할 일 관리**: 일상적인 할 일을 쉽게 추가하고 상태를 관리할 수 있습니다.
- **보상 시스템**: 할 일을 완료하면 캐릭터의 경험치(EXP)를 얻으며, 당일(하루 안)에 완료할 경우 게임 내 상점 이용이 가능한 **포인트**도 추가로 획득합니다.
- **아이템 상점 (Store)**: 모은 포인트를 사용하여 상점에서 무기나 방어구 등 다양한 아이템을 구매하고 캐릭터의 능력치를 강화할 수 있습니다.
- **사냥터 (Hunting Ground)**: 강화된 캐릭터를 이끌고 사냥터로 이동해 몬스터와 전투를 벌이고 경험치를 추가로 얻을 수 있습니다.

## 🛠 기술 스택

- **Core**: React 18, DOM
- **Build Tool**: Vite
- **Language**: TypeScript

## 🚀 프로젝트 실행 방법

이 프로젝트를 로컬 환경에서 실행하려면 **Node.js**가 설치되어 있어야 합니다.

### 1. 의존성 패키지 설치
프로젝트 루트 디렉토리(이 `README.md` 파일이 있는 위치)에서 터미널을 열고 아래 명령어를 실행하여 필요한 패키지들을 설치합니다.

```bash
npm install
```

### 2. 개발 서버 시작
패키지 설치가 완료되면, 아래 명령어를 통해 로컬 개발 서버를 실행합니다.

```bash
npm run dev
```
개발 서버가 시작되면 터미널에 로컬 접속 주소(일반적으로 `http://localhost:5173`)가 표시됩니다. 해당 주소를 브라우저에 입력하여 앱을 사용할 수 있습니다.

### 3. 프로덕션 빌드 및 배포
이 프로젝트는 GitHub Pages를 통해 배포될 수 있도록 설정되어 있습니다. 아래 명령어를 실행하면, 프로덕션 빌드가 자동으로 선행된 후 `gh-pages` 브랜치로 파일들이 푸시되어 배포됩니다.

```bash
npm run deploy
```
배포가 완료되면 `https://cclss-9kiwoon.github.io/todolist-rpg/` 주소에서 라이브 데모를 확인할 수 있습니다.
